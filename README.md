## mitm

mitm is a general purpose man-in-the-middle TLS proxy

## Install mitm
```bash
go get github.com/petethepig/mitm
```

## Usage

```
Usage of mitm:
  -bind-addr string
      bind address (default ":443")
  -ca-cert string
      path to CA certificate path
  -ca-key string
      path to CA key path
```
